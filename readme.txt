== Description ==

Woo order google map location finder helps to find delivery location of ordered items.It is working with WooCommerce only.

It Copy the geolocation of the customer who make order online, So delivary man can easly findout order location.

Features
--------
1. Locate user on google map, by allowing browser to locate
2. user can drag and drop there location on google map
3. Admin can view customer Order delivary location in order detail page
